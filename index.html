🔴 Live mode · Google Sheet polling

Point the dashboard at a published Google Sheet and it will auto-fetch results every few seconds. Multiple people can update the sheet at the same time.

How to set up the sheet (3 steps)
  1. Create a new Google Sheet with this exact header row:
    Type,Name,SNP,LAB,CON,LD,GRN,REF,OTH Example file: live-template.csv in your dashboard folder.
  2. For each declared seat add a row, e.g.
    CON,Glasgow Southside,15422,11000,800,2200,1100,2700,950 Use CON for constituency, REG for region. Names must match the dashboard exactly (e.g. "Glasgow Southside", "Highlands and Islands").
  3. In Sheets: File → Share → Publish to web. Pick the sheet, format Comma-separated values (.csv), click Publish. Copy the URL it gives you and paste it below.
